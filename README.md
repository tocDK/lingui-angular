# @tocdk/lingui-angular

Angular 20+ bindings for [Lingui](https://lingui.dev/) — write English in source, extract via AST, ship PO catalogs.

> **Status: design phase, pre-release.** Not yet installable.
>
> When ready, the library will be installed from this repo directly (not from npm):
>
> ```bash
> npm i github:tocDK/lingui-angular
> ```
>
> The design spec lives at [`docs/superpowers/specs/2026-06-01-lingui-angular-design.md`](docs/superpowers/specs/2026-06-01-lingui-angular-design.md).
